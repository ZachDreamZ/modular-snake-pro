# Release Notes - v0.0.7 - Public Playtest Preview

## 🎮 What's New

### 🎯 Objectives System
Complete 5 objectives (Century, Quarter Millennium, Hungry Snake, Survivor, Chain Reaction) to earn reward points. Track your progress in the new Objectives screen.

### 🏆 3 New Achievements (8 Total)
- **Combo Master**: Reach 10x combo
- **Point Collector**: Accumulate 1000 total points
- **Unstoppable**: Win 3 boss battles

### 💡 Contextual Hints
First-time players now get helpful hints about controls, mechanics, and game modes. Hints appear naturally during gameplay and fade after a few seconds.

### 📊 Persistent Stats
Your progress is now tracked across sessions:
- Games played
- Boss battles won  
- Total food eaten
- Maximum combo reached

### 🎨 Enhanced Menus
New Stats, Achievements, and Objectives screens accessible from the leaderboard.

### ⚖️ Balanced Gameplay
- Weighted food spawning (6 types with configurable probabilities)
- Centralized balancing parameters for easier tuning
- Time Rush HUD with low-time warning

## 🐛 Bug Fixes
- Mode select back button stability improvements
- Shield collision handling in boss battles
- Attribute errors on rapid state transitions

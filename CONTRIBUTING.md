# Contributing to SnakeGradient

Thank you for your interest in contributing! Please read this guide before submitting a PR.

## Development Setup
1. Clone the repo: git clone https://github.com/ZachDreamZ/modular-snake-pro.git
2. Install dependencies: pip install -r requirements.txt
3. Run the game: python main.py

## Running Tests
\\ash
python playtest_bot.py
\All 64 tests must pass before submitting a PR.

## Code Style
- Follow existing patterns in the codebase
- Use type hints where practical
- Keep functions focused and small
- Add comments for complex logic

## Pull Request Process
1. Fork the repo and create a feature branch
2. Make your changes
3. Run tests: python playtest_bot.py
4. Submit PR with description of changes
5. All CI checks must pass

## Reporting Bugs
Use the bug report template and include:
- OS and Python version
- Steps to reproduce
- Expected vs actual behavior
- Screenshots if applicable

## Feature Requests
Use the feature request template. We prioritize:
- Gameplay improvements
- Visual polish
- Accessibility features
- Performance optimizations

Thank you for contributing!
